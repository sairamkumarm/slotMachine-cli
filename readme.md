# SLOT MACHINE - CLI

Built with python, slot machine-cli simulates the actions of a simple slot machine with 3 "slot lines", and was created with the intention to practice python code.

### Winning combinations


| COMBINATION | WINNINGS |
| :-----------: | :--------: |
|     AAA     |    3x    |
|     BBB     |   2.5x   |
|     CCC     |    2x    |
|     DDD     |   1.5x   |
|     AA     |  1.25x  |
|     BB     |    1x    |
|     CC     |  0.75x  |
|     DD     |   0.5x   |
